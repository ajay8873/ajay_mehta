Web Utilities Built for Fun / Practical Solutions

Ward Companion Clinical Utility

An offline-capable clinical toolkit built to support interns, residents, and medical students at the point of care, providing answers within seconds during ward rounds or emergency procedures.

Simple Mechanics

Operates entirely as a Single-Page Application (SPA) with local routing. Caches references and runs fully offline using Service Workers, keeping clinical calculators, reference range tables, and case scripts accessible in hospitals without Wi-Fi.

HTML5/CSS3 Vanilla JS Service Worker PWA jsPDF API ward-companion.pages.dev
💸

Splitway Expense Tracker

A lightweight expense-sharing utility designed to split group costs during trips or shared accommodation, recalculating individual shares and minimizing transactions.

Simple Mechanics

Stores expense records locally in browser memory for offline capability, syncing to a Supabase database when connection returns. Implements a greedy debt-minimization algorithm to recommend simple cash settlements.

Vanilla JS Supabase DB/Auth Local Storage Sync Vanilla Tilt splitway.pages.dev
🎮

Guess the Case Medical Game

A daily diagnostic challenge game where medical students guess clinical diagnoses based on progressive diagnostic clues like patient history, symptoms, and physical signs.

Simple Mechanics

Loads a daily case puzzle and tracks game state (guesses, streaks, and distributions) using local cookies and localStorage. Connects to serverless backend endpoints to fetch percentile rankings and explain wrong choices.

HTML5/CSS3 Vanilla JS Cookies/LocalStorage API Serverless API guessthecase.pages.dev
📚

AJ Medical Archive Study Portal

A structured static repository and digital catalog mapping high-yield study portals and preparation notes for MBBS subjects and PG coaching materials.

Simple Mechanics

Uses JSON data arrays to build searchable indices of flashcards and portal folders, creating a fast, keyboard-navigable static directory that links directly to study folders.

HTML5/CSS3 Vanilla JS JSON Indexing ajmbbsarchive.pages.dev
